## Getting Started

Simple Todo-List write using [NestJS](https://github.com/nestjs/nest) framework, only user with authenticate can access and write Todo.

## Requirements
- NodeJS v16+
- MySql


## Installation

```bash
$ npm install
$ cp .env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
Note : when run command `npm run start` its automatically run migration to database.
